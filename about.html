I created aoe2tournaments in October 2024. I am active in community tournaments, typically rated around 1200 in both 1v1 and team game ELO. I've found that some tournaments can slip between the cracks, and players often only find out about an interesting event after it had started. With this site, I seek to remedy that issue by providing a centralized dashboard for tournaments of all sizes. I am still very new to web development so feel free to make a pull request on the Github if you'd like to lend a hand!
